<h2>
Привет всем! Данный модуль создает событие в гугл календаре использую google calendar api.
Для смены аккаунта, в котором будут создаваться события, воспроизведите следующие действия:
1) Удалите credentials.json и token.json
2) Перейдите по данной ссылке https://console.cloud.google.com/apis/credentials?project=radiant-ion-316118&folder=&organizationId= и создайте oauth2.0 ключ
3) Запустите программу и подтвердите выдачу 

   <img alt="accessibility text" src="https://habrastorage.org/r/w1560/storage/0309cc52/d3bd6b1e/b44ab1f6/64e01804.png" width="100%"/>

Для добавления/изменения комнат измените select с id = room на странице conference.html  
<option value="room id">отображаемое значение</option>
Room id(идентификатор календаря) вы можете найти в настройках своего календаря
